Project Information

  • Client: Huex Inc.
  • Project Type: Group
  • Project Date: January - April 2020
  • Source Code URL: Github

Brief Description

For this project, our team worked with Huex Inc. Toronto to develop an Android application to assist the inventory management process for fast food restaurants. This project's goal is to incorporate the company's existing voice technology into an app that can help managers save time when replenishing and checking their inventories.

Main Features
  • Generate new order
  • View order history
  • Set and check inventory
  • View usage analysis

Technologies Used: Java, Android Studio, NodeJS, MS SQL Server
