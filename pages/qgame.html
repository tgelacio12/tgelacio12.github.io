Brief Description

This QGame application allows users to design and play Q-Game. The main objective of the game is to clear all the boxes by sliding them to doors of the same color. In addition, the user has the option to design their own levels or play a game with the level(s) they have designed.

This app is written in C# and was coded in fulfillment of the requirements for the course PROG2370: Game Programming with Data Structures.

Technologies Used: C#, Visual Studio
